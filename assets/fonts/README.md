# Шрифты

Положи сюда файл фирменного шрифта под одним из имён — он подхватится сам,
править код не нужно (`@font-face` в `index.html` уже описан):

    DrukTextWide.woff2      ← предпочтительно, самый лёгкий
    DrukTextWide.woff
    DrukTextWide.otf
    DrukTextWide.ttf

Достаточно ОДНОГО файла: браузер берёт первый формат, который понимает.

Слово в BRYK лепится из тел по растру глифов, поэтому шрифт влияет на форму
поля, а не только на надпись. По той же причине `font-display:block` — подмена
шрифта на лету пересобрала бы формацию прямо на экране.

Проверить, что шрифт подхватился: открой сцену, добавь `Word · formation`,
в списке шрифтов на карточке должно появиться семейство.
